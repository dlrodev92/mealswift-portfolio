Design Rationale

Key Decisions

Every design choice grounded in UX theory, academic research, and user-centred principles.

🎨
Colour Psychology
Deep navy (#1A3C5E) communicates trust and professionalism, while vibrant orange (#E8630A) stimulates appetite and urgency — directly supporting MealSwift's conversion goals.
#1A3C5E
#E8630A
📱
Mobile-First Responsive
All pages designed for 375px mobile viewport first, scaled to 1440px desktop. Addresses the core issue of non-responsive layout causing mobile drop-offs — aligned with WCAG 2.1 guidelines.
🧠
Nielsen's Heuristics
Navigation redesign applies Nielsen's visibility of system status and user control principles. Real-time order tracking and clear CTAs directly resolve MealSwift's identified UX failures.
👁️
Gestalt Principles
Proximity and similarity used to group restaurant cards and menu items. Consistent visual hierarchy across all 5 pages reduces cognitive load and supports quick decision-making.
🛒
Conversion Optimisation
Sticky cart panel, trust badges, and a prominent Place Order CTA address abandoned cart issues. Two-column checkout layout minimises friction at the most critical conversion point.
♿
Accessibility (WCAG 2.1)
Colour contrast ratios verified for AA compliance. High-legibility sans-serif typography, clear focus states, and descriptive alt text ensure inclusive access for all user groups.
